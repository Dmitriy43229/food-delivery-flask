# 🍕 Food Delivery - Интернет-магазин доставки еды

Веб-приложение интернет-магазина по доставке готовой еды. Покупатели могут просматривать меню, добавлять товары в корзину и оформлять заказы. Администратор управляет заказами и изменяет их статусы.

## 🚀 Технологии

- **Python 3.11** — язык программирования
- **Flask 3.1.3** — веб-фреймворк
- **SQLite** — база данных
- **HTML/CSS** — интерфейс пользователя

## 📋 Функциональность

### Для покупателя:
- Просмотр списка товаров (название, цена, категория)
- Фильтрация товаров по категориям (Пицца, Салаты, Напитки, Десерты, Закуски)
- Добавление товаров в корзину
- Удаление товаров из корзины
- Очистка корзины
- Оформление заказа (имя, телефон, адрес, комментарий)

### Для администратора:
- Вход в защищённую админ-панель (логин/пароль)
- Просмотр всех заказов в виде таблицы
- Изменение статуса заказа: Новый → Готовится → Доставка → Выполнен
- Выход из админ-панели
