## 📖 О проекте

Веб-приложение для заказа готовой еды с доставкой. Покупатели могут просматривать меню, собирать корзину и оформлять заказы. Администратор управляет заказами и отслеживает их выполнение.

**✨ Особенности:**
- 🛒 Корзина на основе сессий
- 🔐 Безопасная авторизация администратора
- 📊 Управление статусами заказов
- 🎨 Адаптивный дизайн

## 🛠 Установка и запуск

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/Dmitriy43229/food-delivery-flask.git
cd food-delivery-flask
2️⃣ Создать виртуальное окружение
bash
python -m venv .venv
3️⃣ Активировать окружение
Windows:

bash
.venv\Scripts\activate
Mac/Linux:

bash
source .venv/bin/activate
4️⃣ Установить зависимости
bash
pip install -r requirements.txt
5️⃣ Настроить переменные окружения
bash
cp .env.example .env
# Отредактируй .env, укажи свой пароль
6️⃣ Запустить приложение
bash
python app.py
7️⃣ Открыть в браузере
